## BGM FACTORY(BGM作成ツール)
ツール概要：
SunoAIで生成したBGMを、
YouTube向け動画として効率的に管理・生成・予約投稿するためのElectronアプリ

## Tech Stack

- Electron
- React
- TypeScript
- FFmpeg
- YouTube Data API
